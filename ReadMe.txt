DEUTSCH:

Der Aufbruch der COVID-19 Daten auf Verbandsgemeinde-Ebene ist aktuell nicht direkt in öffentlich zugänglichen Datenbanken verfügbar. Der Kreis Trier-Saarburg veröffentlicht in einer täglichen Pressemeldung die Daten des Landkreises, aufgebrochen nach Verbandsgemeinden unter www.trier-saarburg.de

Das online Magazin www.5vier.de veröffentlicht diese Pressemitteilungen in einem kontinuierlichen Artikel unter https://5vier.de/corona-update-208649.html

Die Daten für die Verbandsgemeinden, die in diesem Repository gespeichert sind, werden über ein Python Script direkt aus diesem Artikel entnommen.

Die Daten über die Einwohnerzahlen sind über die Bürgerplattform https://www.rlpdirekt.de/ entnommen.

Die Daten werden bereitgestellt wie in dem Artikel aufgelistet, für etwaige Fehler auf der Bereitstellungsseite oder im Python Script, wird keine Haftung übernommen.


Die Daten stellen keine Basis für Interpretationen der aktuell geltenden Bestimmungen im Zusammenhang mit der COVID-19 Pandemie dar und sind nur als Informationsquelle zu betrachten.


Bitte tragen Sie eine Maske in der Öffentlichkeit und halten Sie sich an die entsprechenden Verordnungen der Regierungen und Behörden.

---------------------------------------------------------------------


ENGLISH:

The COVID-19 data on Verbandsgemeinde level for Kreis Trier-Saarburg, are currently not available in any openly accessible database. Kreis Trier-Saarburg is currently publishing the latest figures of the Landkreis as press-statements, split by Verbandsgemeinde on www.trier-saarburg.de

The online magazine www.5vier.de publishes these press-statements in an continuous article: https://5vier.de/corona-update-208649.html

The data for the Verbandsgemeinden, available in this repository, are processed directly from this article.

Data for the number of inhabitants per Verbandsgemeinde is sources via https://www.rlpdirekt.de/

The data is provided as is from the article. No liability is assumed for any errors on the provisioning page or in the Python script.

The data is under no circumstances providing any grounds for interpretation of the currently applicable COVID-19 regulations and shall only be used for information purposes.


Please wear a mask and follow the regulations.
